## UI (User Interface)

A pasta **`ui`** é responsável pela interface de interação entre o usuário e o sistema. Nela estão os componentes que exibem os menus, recebem as entradas do usuário e apresentam os resultados das operações realizadas.

Nesta aplicação, a interface foi desenvolvida em modo texto (terminal), permitindo que o usuário navegue pelas funcionalidades do sistema por meio de um menu simples e intuitivo.

### Arquivos do projeto

* **menu.py:** implementa o menu principal do sistema, exibindo as opções disponíveis e capturando a escolha do usuário para execução das funcionalidades.

### Responsabilidades

A camada de interface é responsável por:

* Exibir o menu principal da aplicação;
* Receber as informações digitadas pelo usuário;
* Encaminhar as solicitações para as camadas responsáveis pelo processamento;
* Apresentar mensagens de sucesso, erro e resultados das operações;
* Facilitar a interação entre o usuário e o Sistema de Gerenciamento de Biblioteca.

Ao separar a interface das demais camadas da aplicação, o projeto torna-se mais organizado, facilitando a manutenção, a reutilização do código e futuras modificações na forma de interação com o usuário.
